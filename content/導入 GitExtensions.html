導入 GitExtensions

為了讓參與 Github 或 Gitlab 協同設計的成員能夠更方便使用 git 指令, 在此導入 https://github.com/gitextensions/gitextensions.

目前推薦使用 GitExtensions 的可攜版本:

GitExtensions.7z 或目前最新版本.

GitExtensions 的 Tools->Settings 包含Git->Path:

必須提供 portablegit 的 bin 路徑以及可攜系統啟動時所使用的 home 位置.

Git Extensions->SSH 則選擇使用 putty

由於 GitExtensions 3.0.2 版尚無法新增提交以中文命名的檔案, 因此必須透過 Tools-> Git GUI 執行新增提交與推送的指令.

使用 Git GUI 的好處是, 使用者可以較方便採用中文說明提交訊息.
